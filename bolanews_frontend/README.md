## Features

- **Home Page**:
  - Menampilkan halaman utama aplikasi dengan informasi terkini.
- **Manage Match**:
  - Menampilkan daftar pertandingan dengan informasi lengkap.
  - Tambah, edit, dan hapus data pertandingan.
  - Tandai pertandingan sebagai "Live".
  - Refresh data pertandingan.
- **Match Detail**:
  - Menampilkan detail pertandingan tertentu.
- **Manage News**:
  - Menampilkan daftar berita.
  - Pencarian berita berdasarkan kata kunci.
  - Tambah, edit, dan hapus berita.
- **News Detail**:
  - Menampilkan detail berita tertentu.
- **Manage Standings**:
  - Menampilkan klasemen liga.
  - Tambah, edit, dan hapus data klasemen.
- **Manage Videos**:
  - Menampilkan daftar video.
  - Tambah, edit, dan hapus video.
- **Video Player**:
  - Memutar video yang dipilih.
- **User Management**:
  - Pendaftaran pengguna baru.
  - Menampilkan daftar pengguna.
  - Halaman profil pengguna.
- **Liga**:
  - Menampilkan informasi liga.
- **Admin Dashboard**:
  - Akses ke fitur manajemen seperti pertandingan, berita, klasemen, dan video.
