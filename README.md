Bola News adalah aplikasi berita sepak bola terkini yang dirancang untuk memberikan informasi terbaru, klasemen, dan video pertandingan kepada penggemar sepak bola di seluruh dunia.

## Installation
### Prerequisites
- Flutter SDK
- Android Studio atau VS Code

### Setup Steps
```sh
# Clone repository
$ git clone https://github.com/yourusername/bolanews.git

# Masuk ke direktori proyek
$ cd bolanews

# Install dependencies
$ flutter pub get

# Jalankan aplikasi
$ flutter run
```
